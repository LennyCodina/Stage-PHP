# Projet PHP – Démarche personnelle de professionnalisation

## Contexte
Ce projet a été réalisé **en parallèle de mes études**, dans une démarche personnelle, afin de démontrer mon **intérêt pour le développement PHP** et mon **investissement continu** en dehors du cadre scolaire.

Il s’appuie sur des ressources pédagogiques OpenClassrooms et vise à mettre en pratique les bases du développement web côté serveur avec PHP et MySQL avec XAMPP.

---
## Technologies utilisées
- PHP
- MySQL
- Apache (XAMPP)
- HTML / CSS
- Git & GitHub

---
## Organisation du projet
Test/
├── includes/ # fichiers images ou autre
├── src/ # contenu principal
├── index.php # fichier principal de liaison
└── README.md # infos sur le projet

---
## Installation (en local)

### Prérequis
- XAMPP installé 
- PHP version 8 ou plus
- Git
### Étapes
1. Cloner le dépôt avec SSH:
```bash
git clone git@github.com:LennyCodina/Stage-PHP.git
```
2. Cloner le dépôt avec HTTPS:
```bash
git clone https://github.com/LennyCodina/Stage-PHP.git
```

---
## Lancement du projet (local)

dans un navigateur:
```bash
http://localhost/test/index.php
```
